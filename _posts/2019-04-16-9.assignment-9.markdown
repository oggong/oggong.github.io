## Ex09_Assignment [대입 연산자]
---
<p>
-대입 연산자<br>
  - 대입연산자: =<br>
  - 축약대입연산자 (산술/ 이진논리 / 쉬프트)<br>
</p>

---

~~~
public class Ex10_Assignment {

	public static void main(String[] args) {

		int a = 10, b = 7;

		a += b; // a = a + b;
		System.out.println("A=" + a);

		a = 10;
		b = 7;

		a -= b;
		System.out.println("A=" + a);

		a = 10;
		b = 7;

		a *= b;
		System.out.println("A=" + a);

		a = 10;
		b = 7;

		a /= b;
		System.out.println("A=" + a);
	}

}

~~~
